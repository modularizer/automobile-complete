This demo shows how easy it is to add autocomplete to any HTML input using the standard CSS overlay pattern. Try typing in the inputs below - you'll see gray suggestion text appear beyond your cursor!

Single-line Input

Type words like "hello", "world", "hi there" to see autocomplete suggestions.

Try it: Type h to see suggestions starting with "h". Press Tab to accept, or use arrow keys to navigate.

Textarea (Multi-line)

Autocomplete works with textareas too! Try typing multiple lines.

Try it: Type w to see "world" or "welcome". Press Enter to create new lines (multi-line mode).

Another Input (Different Styling)

Each input works independently. This one has the same autocomplete but you can style it however you want.

Note: The autocomplete overlay automatically matches your input's styling (font, padding, border, etc.).
